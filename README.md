 ## NestJS Posts API

### Function

- [x] 用户登录验证 ( jwt )
- [x] 用户发帖，增删改查
- [x] 用户对帖子标记收藏
- [x] 用户赞同或反对帖子
- [x] 用户评论帖子
- [x] 数据分页获取

### Usage

1.Clone repository

` git clone https://github.com/111hunter/nest-posts-api.git `

 2.Cd into directory

` npm install `

3.Starting the Server

` yarn start `
